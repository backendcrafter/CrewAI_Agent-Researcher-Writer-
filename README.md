# Crew AI Agents

Building Crew AI agents(researcher & writer). Researcher research for the topic and writer write its summery in simple words to explain you about the topic research.

## How it works
crew_agent.py have both the agents include. Both works together and give us result.

## Tech Stack
- Python
- Groq API (Llama 3.3 70B)
- Crew AI tool

## How to Run
1. Add your Groq API key to .env file: GROQ_API=your_key_here
2. Install: pip install python-dotenv crewai
3. Run python crew_agent.py
